PHP Class for Form Validation.

Tasos Bekos <tbekos at gmail dot com>


Easily extensible and full featured validation library.

Error messages and rules(if needed) can be changed for i18n or other specific project purposes
by extending the default validator class. An example class file for Greek messages exists in folder ext/.

Minimum requirements: PHP 5.3


Based on idea of "Bretticus". See: http://brettic.us/2010/06/18/form-validation-class-using-php-5-3/
